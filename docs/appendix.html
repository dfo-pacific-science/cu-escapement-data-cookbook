12 Appendix

12.1 Reference tables to add

  • Code lists for controlled vocabularies (estimate type, enhancement level, verification status, aggregation flags, benchmark basis).
  • Column dictionary for core tables (CU series, site lookup, metadata, metric specs).
  • Glossary of terms and abbreviations.
  • Mapping of code lists to ontology IRIs where available (stable identifiers for reuse across datasets).

12.2 Case study summaries

  • Condensed summaries of species/area precedents with links back to the source reports for detail.

12.3 External resources

  • Links to the DFO Salmon Ontology site for terminology.
  • Links to canonical code repositories (e.g., WSP-Metrics-Pkg) for metrics and plotting.
  • Links to validation/packaging tools (e.g., Salmon Data Package specs) for automated checks.
