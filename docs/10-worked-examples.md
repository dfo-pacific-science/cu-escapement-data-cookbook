# Worked Examples

## Purpose

Show end-to-end runs that combine the workflow steps, metadata completion, and
metric execution.

## Examples to build out

- Yukon Chinook (e.g., Stewart CK-74) run-reconstruction example: inputs,
  control files, outputs, and how they map to CU series and metadata.
- Fraser Chinook nuSEDS direct-sum example: site selection, record processing,
  aggregation, metadata linkage, and metrics run.

## What each example should contain

- Input files and specs used
- Step-by-step commands or scripts
- Key decisions and justifications
- QC plots or tables and how to interpret them
- Final CU series, metadata, and metrics outputs
- A comparison against the previous release, with a changelog explaining
  differences
